nomumo



Was ist nomumo?

nomumo ist eine Open-Source-Alternative zu Terminplanern wie z.B. doodle.com.

Implementierung

Frontend

Das Frontend verwendet moderne Technologien wie beispielsweise:
  • HTML5
  • JavaScript (Framework: AngularJS)
  • CSS (Bootstrap)

Backend

Alle Daten werden in einer MySQL-Datenbank gespeichert, verwaltet und stehen über eine RESTful-API einer NodeJS-Applikation zur Verfügung. Neben der API hostet NodeJS die eigentliche Website. Als SMTP-Server verwendet nomumo.de eine Microsoft-Exchange-Instanz. Der Exchange-Server kann jedoch beliebig geändert werden.
